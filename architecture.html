System Architecture

SmartGradeAI Components & Data Flow

1. Users
Teacher
  • Create & manage courses
  • Upload exam PDFs
  • Upload grading rubric PDFs
  • View AI-graded student scores
Student
  • Enroll in available courses
  • Submit solution PDFs
  • View automated AI feedback
  • Check overall grade & metrics
→
2. Client Interface
React Application
  • Framework: Vite + React.js
  • Styling: Tailwind CSS
  • Views: Role-based Dashboards
  • State: File uploads & Auth context
  • Integration: Axios REST calls
↔
3. Backend Services
FastAPI Server
  • API: RESTful Python Endpoint
  • Auth: JWT & Password Hashing
  • PDF Ops: Extract text & local storage
  • Logic: Custom Prompt Engineering
↕
Storage & Database
  • Database: SQLite Relational Data
  • ORM: SQLAlchemy (Users, Grades)
  • Files: Local `/uploads` directory
↔
4. AI Provider
Google Gemini API
  • Input: Extracted Student PDF + Teacher Rubric PDF
  • Task: LLM Comparative Evaluation
  • Format: Structured JSON Responses
  • Output: Automated Final Score & Constructive Feedback
